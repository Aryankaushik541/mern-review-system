# MERN Review System

Full-stack review management system with authentication.

## Setup

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Edit .env file
npm run dev
```

### Frontend
```bash
cd client
npm install
npm start
```

## Tech Stack
- MongoDB, Express, React, Node.js
- JWT Authentication
- Role-based access (User/Admin)
